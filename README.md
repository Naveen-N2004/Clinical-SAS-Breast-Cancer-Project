# Clinical-SAS-Breast-Cancer-Project
Clinical SAS project using SDTM, ADaM, TLF and mutation analysis in breast cancer dataset.
Breast Cancer Mutation Analysis using Clinical SAS
Project Overview

This project focuses on mutation analysis in breast cancer clinical data using Clinical SAS programming techniques. The workflow includes raw data import, SDTM domain creation, ADaM dataset generation, statistical analysis, visualization, and TLF (Tables, Listings, Figures) reporting.

The project was developed as part of hands-on Clinical SAS practice to understand real-world clinical trial programming standards and data analysis workflows.

Objectives
Import and validate raw clinical datasets
Perform QC (Quality Check) procedures
Create SDTM domains:
DM (Demographics)
MH (Medical History)
EX (Exposure)
LB (Laboratory)
TU (Tumor Identification)
SC (Subject Characteristics)
Create ADaM analysis datasets
Perform mutation and TMB analysis
Generate Tables, Listings, and Figures (TLF)
Export analysis report into PDF format
